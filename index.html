マップ読み:手元に集中しながら敵の位置を把握

遊び方 下のフィールドの数字インクを1から順番にタップ(=塗り・撃ち合いの代わり。集中を持っていかれる作業役)。最後まで押し切ると新しい配置でループ。 その間、上のイカランプで味方(緑)・敵(ピンク)の生死=人数有利/不利が変化し、黄色く光る人はSPがたまっている合図。 ミニマップは左(青)・中(灰)・右(紫)で色分け。生きているイカが動き、ときどき潜伏(消える)します。 突然クイズだけの画面になり「人数は有利?不利?」「敵/味方はどこに多い?」「SPがたまってる人は何人?」等を聞かれるので、 手元をやりながらチラ見で盤面を覚えて即答(クイズ中は盤面が止まります)。実戦の「塗りながら盤面を読む」神経を鍛えます。正解率80%目標。

ヘルプ・使い方

このアプリは?

スプラの上達を「1日1課題・10秒記録・盤面を読む練習」で支える個人用アプリ。勝敗より課題をこなせたかで自分を伸ばします。データは全部この端末の中だけ(誰にも送られません)。

スマホに「アプリ」として入れる
  • iPhoneSafariでこのページを開く → 共有ボタン(□に↑) → 「ホーム画面に追加」
  • AndroidChromeで開く → メニュー(⋮) → 「アプリをインストール」
追加すると専用アイコンで全画面起動・オフラインでも動きます。
友達に渡す

このページのURLをLINE等で送るだけ。相手も同じ手順でアプリにできます。戦績は各自の端末ごとに貯まり、混ざりません。

各タブの使い方
  • 🏠ホーム今日の課題を確認・選ぶ・自分で追加
  • ✍️記録試合後にWIN/LOSEを押して10秒入力
  • 📊分析直近50戦の弱点を確認。AIコーチ用コピーもここ
  • 🗺️マップ読み手元をやりながら盤面を読む練習。難易度3段階。プレイ中は「中断する」で停止
難易度の変え方

「🗺️マップ読み」タブのスタートボタン上にあるやさしい / ふつう / むずいをタップ。選んだ難易度は記憶されます。まずは「やさしい」で正解率80%を目標に。

AIにコーチしてもらう

「📊分析」タブの「📋分析データをコピー」を押す → ChatGPTやClaude等に貼り付けて送信 → 弱点と次の課題を提案してくれます。

困ったとき
  • 更新されないアプリを完全に閉じて開き直す(1回でOK)
  • 記録を全部消したい「📊分析」タブ一番下の「記録をすべてリセット」
  • データは消える?普通は消えません。iPhoneは長期間(目安7日)一度も開かないと消えることがあるので週1は起動を
